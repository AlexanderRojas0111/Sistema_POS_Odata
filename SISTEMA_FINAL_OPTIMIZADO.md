## 🚀 **CÓMO INICIAR EL SISTEMA**

### **Opción 1: Script Automatizado (Recomendado)**
```powershell
.\start_sabrositas.ps1
```

### **Opción 2: Docker**
```powershell
.\docker-start.ps1
```

### **Opción 3: Manual**
```powershell
# Backend
.\venv_python313\Scripts\Activate.ps1
python main.py

# Frontend (nueva terminal)
cd frontend
npm run dev
```
